## What to do before commercial distribution

For each PDF marked ⚠️ above, the author must :

1. Identify the actual source (self-authored, scraped, downloaded, purchased).
2. Check the license / terms (CC, proprietary, private research, etc.).
3. Decide :
   - ✅ **Keep bundled** if license allows redistribution.
   - 🔗 **Replace with URL citation** if license allows reference only — delete the PDF, document the URL here and in the bibles.
   - ❌ **Remove entirely** if unclear or proprietary — delete the PDF, paraphrase the learnings in the bibles (which is already done at 95%).

When in doubt : **remove the PDF and keep only the URL citation.** The bibles already internalize the content.
